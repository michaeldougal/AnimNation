Spring

A physical model of a spring for Roblox. Based off of Quenty's open source version (https://github.com/Quenty/NevermoreEngine/blob/main/src/spring/src/Shared/Spring.lua)

Visualization (by Defaultio): https://www.desmos.com/calculator/hn2i9shxbz
